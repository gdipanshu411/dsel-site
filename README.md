# The Dream (DSEL) — Sidhi

Static site for The Dream Spoken English Institute, Sidhi.

## Pages
- index.html — Home
- about.html — About
- courses.html — Courses
- activities.html — Activities
- news.html — News

- tests.html — Tests & assessments
